# Preferences

This document describes the runtime preferences stored locally for Kakeibo.

- **Base currency**: the default currency used for display and planning.
- **Locale**: locale used for date and number formatting.
- **Week starts on**: the first day of week (0 = Sunday, 1 = Monday).

Preferences are persisted in IndexedDB under the `appSettings` table and are editable from the `Settings` screen.
